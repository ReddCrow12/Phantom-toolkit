Phantom Hacking Tool

Description:
Phantom is a powerful hacking tool designed for ethical hacking and cybersecurity purposes. It provides functionalities such as network scanning, exploitation of vulnerabilities, and more.

Installation and Usage:
Requirements:
- Python 3.x
- Libraries: socket, os, sys

Installation:
1. Clone the repository:
   $ git clone https://github.com/your/repository

2. Navigate to the Phantom directory:
   $ cd Phantom

Usage:
1. Open a terminal or command prompt.

2. Navigate to the Phantom directory:
   $ cd path/to/Phantom

3. Run Phantom:
   $ python phantom.py

Commands:
- help: Display help information.
- -a: Run automated attack module.
- connect <port> <IP>: Connect to a target using a specified port and IP.
- scantom <IP>: Perform network scanning for vulnerabilities.
- exit: Exit Phantom tool.

Examples:
- Connect to a target:
  $ Phantom>: connect 22 127.0.0.1

- Run a network scan:
  $ Phantom>: scantom 192.168.1.1
